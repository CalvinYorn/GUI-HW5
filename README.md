Scrabble Game by Calvin Yorn - HW5

Email: calvin_yorn@student.uml.edu

Features:

- Full 15x15 Scrabble board with all bonus squares
- Drag and drop tiles from rack to board
- Drag tiles back from board to rack
- Tile adjacency validation
- Accurate scoring with multipliers
- Recall Tile button to undo placements
- Refresh Tiles button to swap tiles
- Letter counter showing remaining tiles
- New Game button to reset everything
- Validate Word button to score and deal new tiles

How to Play:

1. Drag tiles from your rack onto the board
2. Click Validate Word to score your word
3. New tiles are automatically dealt
4. Click New Game to start over

Button Functions:

Validate Word - Scores your word with bonus multipliers and deals new tiles

New Game - Resets board, score, and tile distribution

Recall Tile - Returns last placed tile to your rack

Refresh Tiles - Exchanges all tiles for new random ones
